# Teslo API

1. Clonar proyecto
2. Instalar dependencias ```npm install```
3. Clonar el archivo ``` .env.example``` y renombrarlo a ```.env```
4. Llenar las variables de entorno
5. Levantar la base de datos
```
docker-compose up -d
```
6. Levantar: ```nest start --watch ```
